    Quantitative Economics with Python is an on-line course on quantitative economic modeling using the Python programming language, authored by John Stachurski and Thomas J. Sargent.

    The course consists of

    • A series of lectures
    • A code repository containing Python programs described in the lectures

    Getting Started

    The lecture series provides detailed information on getting starting with Python and working with the code repository.

    As discussed in the lectures, the code repository is stored on Github, the primary function of which is to host open source code repositories in a manner that facilitates sharing and collaboration. If you simply wish to download a copy of the repository you can do so by clicking the 'Zip' (or 'Tar') button above. If you want to learn a bit more about Git and Github---you'll find it's an excellent investment---then start here.
